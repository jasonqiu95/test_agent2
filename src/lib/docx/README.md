# DOCX Parser Module

A TypeScript wrapper around [mammoth.js](https://github.com/mwilliamson/mammoth.js) for parsing Microsoft Word `.docx` files.

## Features

- Convert DOCX files to HTML
- Extract plain text from DOCX files
- Full TypeScript support
- Clean, promise-based API
- Comprehensive error handling

## Usage

See the example file at `src/examples/docx-parser-demo.ts` for usage examples.
